# Frontal CLI (Homebrew Tap)

Homebrew tap for the [Frontal CLI](https://github.com/frontal-labs/cli).

## Usage

```bash
brew tap frontal-labs/cli
brew install frontal-cli
```

## Updating

```bash
brew upgrade frontal-cli
```

## Uninstalling

```bash
brew uninstall frontal-cli
brew untap frontal-labs/cli
```
